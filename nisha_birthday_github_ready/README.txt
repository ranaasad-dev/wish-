NISHA BIRTHDAY WEBSITE - GITHUB PAGES READY

1. Create a new PUBLIC GitHub repository, for example: nisha-birthday
2. Open the repository and choose: Add file > Upload files
3. Upload ALL files from this folder separately.
4. Do not upload only the ZIP file.
5. Make sure index.html is visible at the root of the repository.
6. Go to Settings > Pages.
7. Under Build and deployment choose:
   Source: Deploy from a branch
   Branch: main
   Folder: /(root)
8. Click Save and wait a few minutes.

Expected link:
https://YOUR-USERNAME.github.io/nisha-birthday/

The photos are included and already connected to the website.
